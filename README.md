# Robust Experiments

This repository contains the code used to reproduce the simulation and empirical results in the paper:

**Residual Propagation and Consistent Estimation in Multivariate Time Series**

---

## Overview

This project studies estimation in time series models under contamination.  
The code implements simulation experiments and empirical illustrations designed to evaluate the effect of contamination and the proposed patch removal procedure.

The focus is on **estimation**, not on outlier detection.  
In particular, the simulation design assumes that contaminated observations are known (oracle setting), so that the impact of contamination propagation can be isolated.
